    ez-sprites

    Minimalist JS library for creating simple 2D graphical games. Sprites are objects with simple properties: x/y coordinates for position, and dx/dy values for movement. Also supports SVG files for sprites.

    • Sprite Composer

    A companion tool for ez-sprites which makes designing "compound-shape sprites" easy. This is mostly obsolete since SVG is a much better way to design graphics, but it was helpful for a time and remains a good way to demonstrate objects and properties. The JSON is live-editable. I began work on a similar tool for pzsprites to help make bounding boxes for SVG sprites; it's in the pzsprites repo.
