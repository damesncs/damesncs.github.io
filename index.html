Tools

Konsol

Konsol - minimalist browser JS environment with text input/output. Includes JavaScript Reference page with example programs. Most of the examples are the basis of code-along lab exercises.

Labs

These are in-browser "laboratories" to explore concepts.
  • Image Lab - image manipulation toolkit
  • Crypto Lab - public-key cryptography demo
  • Bitmap Font Designer - explore encoding by creating a bitmap typeface
  • Curve Lab - very simple demo of quadratic and cubic Bezier curves - click and drag

    • Libraries

    ez-sprites

    Minimalist JS library for creating simple 2D graphical games. Sprites are objects with simple properties: x/y coordinates for position, and dx/dy values for movement. Also supports SVG files for sprites.

    • Sprite Composer

    A companion tool for ez-sprites which makes designing "compound-shape sprites" easy. This is mostly obsolete since SVG is a much better way to design graphics, but it was helpful for a time, helped me learn a lot, and remains a good way to demonstrate objects and properties. The JSON is live-editable. I began work on a similar tool for pzsprites to help make bounding boxes for SVG sprites; it's in the pzsprites repo.
